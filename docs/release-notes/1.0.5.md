HowLong 1.0.5

What is new:
- Added haptic feedback across key actions so creating, updating, logging, and deleting feel more responsive.
- Extended haptics to smaller interactions like tabs, quick log buttons, chips, pickers, and reminder controls.
- Improved the habit reminder input so tapping the bell area or reminder row opens the time picker, not just the visible time.

Fixes and improvements:
- Refined reminder interaction on the habit form for a larger, clearer tap target.
- Kept destructive actions using stronger feedback while lighter selections stay subtle.
