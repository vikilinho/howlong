HowLong 1.0.3

What is new:
- Improved Android reminder reliability for actions and habits.
- Added support for exact alarm scheduling when the device allows it.
- Hardened save flows so actions and habits no longer get stuck on a loading state if a save step fails.

Fixes and improvements:
- Fixed a release-only Android notification issue where scheduled reminders could fail to appear.
- Fixed a release receiver crash caused by code shrinking during Play Store style builds.
- Cleaned up temporary debug tracing added during production diagnostics.
